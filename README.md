### 기능 
#### 1) 읽고 있는 책 검색
- 네이버 도서 검색 API 활용
- 검색 결과 리싸이클러뷰에 담기

#### 2) 읽고 있는 책 관리
- 멀티 뷰 홀더 리싸이클러뷰
- 노트 기능
  - 노트 저장, 수정, 삭제
  - 카메라 / 갤러리 이미지 업로드
- 타이머 기능

#### 3) 다 읽은 책 관리
- 읽고 있는 책에서 '다 읽은책으로 등록' 버튼을 누르면 다 읽은 책 관리 목록으로 넘어 감
- 별점, 리뷰 작성 가능

#### 4) 차트
- 연/월 별 읽은 책 권 수, 독서 시간(타이머 측정)을 차트로 표현
